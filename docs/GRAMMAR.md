## 1. What Brajawali is

Brajawali (also written *Brajbuli*) is a **constructed literary language** — not anyone's mother tongue, not the spoken Braj of Mathura. Sankaradeva built it deliberately for his *Ankiya Nat* (one-act plays), *Borgeet* (devotional songs), and *Bhātimā* (praise poems) so they would be understood across North India in the 15th century. The vocabulary is a stylised mix of:

- **Maithili** (the metrical backbone)
- **Sanskrit** (philosophical and religious terms)
- **Avahaṭṭha** / Apabhraṃśa (verb endings, postpositions)
- **Assamese** (local idiomatic words, jatuwa thaach)

Goswami calls the Assam variant specifically "**অসমীয়া ব্ৰজাৱলী**" because its phonology is Assamese (e.g. `স` is pronounced `s`, not `ʃ` like in Bengali Brajbuli) and many Assamese vocabulary items are embedded.

The corpus is **closed** — only ~3,017 distinct words attested across the canonical 13 plays + Borgeet + Bhatima. Modern Assamese terms (ফোন, কম্পিউটাৰ) have no Brajawali equivalent.

## 2. The conversion mindset

Don't translate word-by-word from English to Brajawali. Instead:

1. **Start with modern Assamese.** What would you naturally say?
2. **Replace the function words** (pronouns, postpositions, verb endings) with the Brajawali equivalents — these are highly regular.
3. **For content words** (nouns, adjectives, verbs), pick the closest Brajawali synonym from the lexicon.
4. **Apply biprokorṣa** to dissolve any Sanskrit conjuncts that survive.

Word order stays Assamese SOV — Brajawali doesn't reorder. Only morphology and lexicon change.

## 3. Alphabet and phonology

Brajawali uses the same Bengali-Assamese script as Assamese, with **47 letters**:

- **11 vowels**: অ আ ই ঈ উ ঊ ঋ এ ঐ ও ঔ
- **36 consonants**: ক খ গ ঘ ঙ চ ছ জ ঝ ঞ ট ঠ ড ঢ ণ ত থ দ ধ ন প ফ ব ভ ম য ৰ ল ৱ শ ষ স হ ক্ষ ং ঃ

9 letters lack independent use (ঊ ঔ ঋ ঞ ণ ৱ ষ ং ঃ) — they only appear inside words.

**Pronunciation conventions** to keep in mind:

| Letter | Brajawali sound | Note |
|---|---|---|
| স | s | Same as Assamese — **not** the Bengali `ʃ` |
| চ | s | Same as Assamese (palatal-affricate-turned-fricative) |
| ৰ | r (alveolar tap) | Assamese ৰ, not Bengali র |
| ৱ | w | Like Assamese ৱ |
| য | y / j | Often written য়, pronounced y- initial |

## 4. Biprokorṣa — cluster-breaking rules

Brajawali systematically **dissolves Sanskrit consonant clusters** by inserting epenthetic vowels (mostly `অ` or `উ`). This is the single most distinctive phonological feature.

### Core rules (~110 documented in Goswami Ch. 2-ঘ)

| Sanskrit cluster | Brajawali | Example |
|---|---|---|
| গ্নি | গনি | অগ্নি → অগনি (fire) |
| জ্ঞ | গিয় | অজ্ঞান → অগিয়ান (ignorance) |
| স্হ | থ | স্হিৰ → থিৰ (firm) / স্হান → থান (place) |
| ক্ষ | খ | ক্ষীৰ → খীৰ (milk) / ক্ষত → খত (wound) |
| স্ম | সুম | স্মৰণ → সুমৰন (memory) |
| স্ত্ৰ | তৰ | স্ত্ৰাস → তৰাস (fear) |
| স্নান | চিনান | স্নান → চিনান (bath) |
| ক্ত | কত | ভক্ত → ভকত / মুক্ত → মুকুত |
| ক্তি | কতি | ভক্তি → ভকতি / মুক্তি → মুকুতি / শক্তি → শকতি |
| ক্ৰ | কৰ | চক্ৰ → চকৰ (wheel) |
| প্ৰ | পৰ | প্ৰচাৰ → পৰচাৰ / প্ৰভু → পহু / প্ৰণাম → পৰণাম |
| প্ৰে | পৰে | প্ৰেম → পেম / প্ৰীতি → পীৰিতি |
| ৰ্থ | ৰথ | অৰ্থ → অৰথ |
| ৰ্ম | ৰম | ধৰ্ম → ধৰম / কৰ্ম → কৰম / মৰ্ম → মৰম |
| ৰ্ণ | ৰণ | বৰ্ণ → বৰণ |
| ৰ্ব | ৰব | দৰ্ব → দৰব / গৰ্ব → গৰব |
| গ্ম | গম | মগ্ন → মগন |
| ত্ত | ত | চিত্ত → চিত |
| দ্ব | দব | দ্বাদশ → দবাদশ |
| ভ্ৰ | ভৰ | ভ্ৰম → ভৰম |
| ৃ | উৰু | মূৰ্খ → মূৰুখ / মূৰ্ত্তি → মূৰুতি / সূৰ্য → সূৰুজ |

### র / ল interchange

Sanskrit's *ralayoṛabhedaḥ* rule (ৰ and ল are interchangeable) applies extensively:

| Form A | Form B |
|---|---|
| খেলা | খেৰা |
| কেলি | কেৰি |
| গালি | গাৰি |
| কোল | কোৰ |
| পোলে | পোৰে |
| নাবোল | নাবোৰ |

### Borṇa-bipordzaya (sound-shifts)

| Assamese | Brajawali |
|---|---|
| চকু | আখি |
| মুখ | মুহ |
| হাত | হাথ |
| শ্ৰী | ছিৰি |
| ৰক্ত | ৰাতা |
| হৰষে | হৰথে |

### First-person pronoun nasalisation

| Assamese | Brajawali |
|---|---|
| মই | মঞি |
| তই | তঞি |
| তেওঁ | তেঞো |
| যাও | যাঞু |
| খাও | খাঞু |
| লওঁ | লিঞ্তো |
| নাই | নাঞি |

## 5. Pronouns

**Most striking feature: pronouns are gender-neutral and honorific-neutral.**

### First person

| Modern Assamese | Brajawali | Notes |
|---|---|---|
| মই | **মঞি**, হামো, হামি | hāmo is most common in songs |
| আমি | **হামো**, হামোসব | plural via *সব* |
| মোক / আমাক | **হামাক** | accusative |
| মোৰ / আমাৰ | **হামাৰ**, হামাৰি, হামাৰু | genitive |
| মোৰে (instr.) | হামাৰে | |
| মোত | হামাত, হামাতে | locative |

### Second person — collapses three honorific levels

| Modern Assamese | Brajawali |
|---|---|
| তই / তুমি / আপুনি | **তোহো**, তঞি, তুমহি |
| তোমাক / আপোনাক | **তোহাক**, তোহে |
| তোমাৰ / আপোনাৰ | **তোহাৰ**, তোহাৰি |
| তোমালোক | **তোহোসব** |

This means *"হামাৰ, তোহো"* covers respectful and casual speech identically.

### Third person — gender-neutral

| Modern Assamese | Brajawali |
|---|---|
| সি / তাই / তেওঁ | **তেহো**, তেঞো |
| তাক / তাইক / তেওঁক | তাহাক, তাহেক |
| তাৰ / তাইৰ / তেওঁৰ | তাহাৰি, তাহেৰি, যাহেৰ |
| তেওঁলোক | তেহোসব, তাৰাসব |
| সেইটো | সোহি, ওহি, সো, সে |
| এইটো | এহু, এ, এই |

### Demonstratives

| | Proximal (this) | Medial / Anaphoric | Distal (that) |
|---|---|---|---|
| Singular | এহু, এই, এ | সে, সো, সোহি | ওহি, ও |
| "Such" | এঁচন, এছে | তেছন, তৈচন | — |

### Interrogatives

| What | Where | When | How | Why |
|---|---|---|---|---|
| কি / কী / কাহা / কমন | কথি / কাহা / কোথি | কব / কবহু | কৈচে / কেনে | কাহে |

### Relatives

| যি (rel.) | তো (corr.) |
|---|---|
| যো / যাহে | তো / তাহে / সো |
| যব (when) | তব (then) |

## 6. Cases (the 7 vibhakti)

Brajawali keeps Assamese's 7 cases but uses **different markers**. Master these and 80% of conversion is done.

| Case | Modern Assamese marker | Brajawali marker(s) | Example (Assamese → Brajawali) |
|---|---|---|---|
| **Nominative** | (zero) / -এ | (zero) / -অয়ে / -ই / -এ | মই → মঞি |
| **Accusative** | -ক | **-ক / -কু / -কে** | তোমাক → তোহাক |
| **Instrumental** | -এৰে | **-এ / -এৰ / -ৰে / -য়ে** | হাতেৰে → হাথেৰ |
| **Dative ("to")** | -লৈ | **-কলাগি / -এলাগি / -কুলাই / -ৰে / -মহ** | ঘৰলৈ → ঘৰকলাগি |
| **Ablative ("from")** | -পৰা / -ৰ পৰা | **-হন্তে / -তহন্তে / -ত / -ৰ** | আকাশৰ পৰা → আকাশহন্তে |
| **Genitive** | -ৰ | **-ক / -কেৰি / -কৰ / -ৰ** | কৃষ্ণৰ → কৃষ্ণক / কৃষ্ণকেৰি |
| **Locative** | -ত | **-ত / -তে / -এ / -মহ** | ঘৰত → ঘৰে / গৃহে |

> ⚠ **The most common mistake**: using `-ৰ` as genitive in Brajawali. Use `-ক` or `-কেৰি` instead. *কৃষ্ণৰ* in Brajawali is *কৃষ্ণক*, not *কৃষ্ণৰ*.

### Vocative (a 9th category)

For addressing someone:

| Assamese | Brajawali particles |
|---|---|
| ও / এৰা / হেৰা | এ, ঞঁ, অয়ে, অৰে, আগো, আলো, আহে, ওৰে, ৰে, হে, হেৰি |

Examples:
- *এ মাঞি* (O mother)
- *আহে সখী* (O friend)
- *ওৰে সখি পেখোৰে* (O friend, look!) — Borgeet

## 7. Verbs and tense

Brajawali has **three tenses** (present, past, future) but **9 verb classes** based on aspect/mood. Endings come from Avahaṭṭha/Apabhraṃśa, not Sanskrit.

### Present — habitual / nityo-bortomān

| Person | Modern Assamese | Brajawali |
|---|---|---|
| 1sg | কৰোঁ / খাওঁ | কৰো / খাঞু (যাঞু = "I go") |
| 2sg | কৰা / খোৱা | কৰস / খাৱত |
| 3sg | কৰে / খায় | কৰে / কৰয় / কৰয়ে / খাই |

### Present — continuous / swarūp-bortomān

| Person | Modern Assamese | Brajawali |
|---|---|---|
| 1sg | কৰিছোঁ | কৰৈছি / কৈছোঁ |
| 2sg | কৰিছা | কৰৈছ |
| 3sg | কৰিছে | কৰৈছে / কৈয়া আছে |

### Past — perfective / swarūp-bhūt

The signature Brajawali past is **stem + -ল- + person**.

| Person | Modern Assamese | Brajawali |
|---|---|---|
| 1sg | কৰিলোঁ | **কয়লো / কৰলো** |
| 2sg | কৰিলে / কৰিলি | **কয়লি** |
| 3sg | কৰিলে | **কয়ল / কহল** |

So the simple past pattern:
- *আনিলোঁ* → **আনলো** ("I brought")
- *গ'ল* → **গেল**
- *দেখিলোঁ* → **দেখলো / পেখলো**
- *আহিলোঁ* → **আইলো**

### Past — pluperfect / apurṇo-bhūt

For "had done":

| Person | Brajawali |
|---|---|
| 1sg | কয়েছিলো / থৈয়াছিলো |
| 2sg | আছিলা / কহৈছে |
| 3sg | খেলাইছিল / পেখল |

### Future / bhabiṣyat

Stem + **-ব** + person. Also used for command, blessing, request.

| Person | Modern Assamese | Brajawali |
|---|---|---|
| 1sg | কৰিম | **কৰব** |
| 2sg | কৰিবি / কৰিবা | **কৰবি / এৰাইবা** |
| 3sg | কৰিব | **কৰব / আনব** |

### Imperative / anuğyā

Distinguishes mānyārtho (polite) and tucchārtho (familiar):

| Form | Brajawali |
|---|---|
| 2nd polite | আনহ, কৰহ, উঠ, শূনলো, আলিঙ্গহ |
| 2nd familiar | কৰহ, চড়সিয়া, চোড়হ, কহগিয়া, বোলগিয়া |
| 3rd polite | কৰতু, জপহু, ৰহু |
| 3rd familiar | দেখোক, পেখোক |

The **-হ ending** is the most common Brajawali polite imperative.

### Conjunctive participle / asamāpikā

For "having done X" — endings: *-ই, -ইয়া, -ইয়ে*.

- *পেখি* (having seen)
- *জানিয়া* (having known)
- *আকুলি* (having become agitated)
- *হেৰিয়ে* (looking)
- *গৈয়া কহো* (having gone, said)

### Infinitive purposive / tum-arthak

For "in order to":

- *কদাৰ্থতে* (in order to slay)
- *পলাইতে* (in order to flee)
- *গিলিতে* (in order to swallow)
- *কৰিতে লাগিয়ে* (begins to do)

### Causative / pāchanī

Stem + **-আৱ-**:

| Verb | Causative |
|---|---|
| কৰ → কৰাৱ | কৰাৱে (he causes) / কৰায়া থিক / কৰাৱত |
| অন → অনাৱ | অনাৱলো (I had brought) |
| দেখ → দেখাৱ | দেখাৱলো (I showed) |
| পৰ → পৰাৱ | পৰাৱল (you made fall) |

### Continuous-simultaneous / śatū-bodhok

Often reduplicated:

- *কান্দিতে কান্দিতে* (weeping-weeping)
- *হাসিতে মাতিতে* (laughing-talking)
- *খোজগুড়ি যাইতে* (while crawling-going)
- *কৰইতে* (while doing)

## 8. Negation

Three patterns; pick by position.

| Pattern | Example | Usage |
|---|---|---|
| **নাহি ___** | *নাহি দেখো* (I do not see) | most common |
| **ন___** | *নকৰহ* (don't do) | imperatives |
| **___বি নাহি** | *চাড়বি নাহি* (do not let go) | future-prohibitive |
| **নকৰহ / নযায়ো** | inverted | colloquial |
| **নোহে** | "is not" | copula negation |

Modern Assamese:
- *নকৰোঁ* → **নাহি কৰো** or **নকৰোঁ → নাহি কৰব**
- *নকৰিবা* → **কৰবিনা** / **নকৰহ**
- *নাজানো* → **নাহি জানো** / **নজানি**

## 9. Numbers and pluralisation

### Plural

Brajawali has **no -বোৰ / -সকল suffix**. Pluralisation is *periphrastic* — append a separate word:

| Plural marker | Use |
|---|---|
| **সব** | most common: *গোপীসব, তোহোসব, ছাত্ৰসব* |
| সকল | *গোৱালসকলে* |
| গণ | *দেৱতাগণ, গোপগণ* |
| লোক | *ৰাজলোক* |
| নিকৰয়ে | *বালা গাৱেনিকৰয়ে* |
| কুল | *অলককুল* |

### Numbers

| Modern | Brajawali |
|---|---|

| দুই | দোহো |
| তিনি | তিনি / তিন |

## 11. Vocatives and interjections

| Function | Brajawali |
|---|---|
| general "O / Hey" | এ, ঞঁ, ওৰে, ৰে, হে |
| affectionate (to female) | আগো, আলো, হেৰি |
| respectful | আহে, অয়ে |
| excited | হুৰ, পেখোৰে |
| pain / lament | আঃ, হা হা, ছিঃ, হা হী |
| approval | আৰৰ, ই |

Idiomatic addressee phrases:
- *এ মাঞি* — "O mother"
- *আহে সামাজিক লোক* — "O assembled people" (used in nat openings)
- *ওৰে সখি পেখোৰে* — "O friend, look!"
- *ৰে ৰে পুৰম্দৰ* — "Hey hey, Indra!"

## 12. Compound words (samāsa)

Six types from Sanskrit; Brajawali uses them sparingly. Most "compounds" in Brajawali texts are pure Sanskrit imports — Goswami warns against treating them as native.

| Type | Definition | Brajawali example |
|---|---|---|
| **Karmadhāraya** | adj + noun, adj qualifies | চান্দমুখী = চান্দ-তুল্য মুখ-ধাৰিণী ("moon-faced") |
| **Dvigu** | numeral compound | মাস-চাৰিকৰ ("of four months"), চৌদিশে ("in all four directions") |
| **Bahuvrīhi** | exocentric (refers outside) | পীতপিচোড়া ("[he] of the yellow cloth" = Krishna) |
| **Dvandva** | coordinative | অবনিসি (অব + নিসি = "day and night"), দিবসৰয়নি |
| **Avyayībhāva** | indeclinable + noun | অবিঘিনিয়ে, প্ৰতি দিনে দিনে |
| **Tatpurusha** | case-determined | (folded into nirbibhaktika declension) |

## 13. Tadbhava ↔ Tatsama

Words come into Brajawali in two flavors:

- **Tatsama** — direct from Sanskrit, unchanged form: চক্ৰ, ধৰ্ম্ম, প্ৰচাৰ
- **Tadbhava** — Sanskrit-derived but phonologically dissolved: চকৰ, ধৰম, পৰচাৰ

In practice Brajawali **prefers tadbhava** — apply biprokorṣa wherever possible. When a Sanskrit form lands in Brajawali, you typically get its tadbhava counterpart in songs and dialogue, with the tatsama reserved for ślok-style emphatic verses.

| Sanskrit (tatsama) | Brajawali (tadbhava) |
|---|---|
| অগ্নি | অগনি |
| চক্ৰ | চকৰ |
| তীৰ্থ | তীৰ্থি |
| দৰ্প | দৰপ |
| ধৰ্ম্ম | ধৰম |
| প্ৰচাৰ | পৰচাৰ |
| বুদ্ধি | বুধি |
| ভক্তি | ভকতি |
| মুক্তি | মুকুতি |
| স্হান | থান |
| স্নান | চিনান |
| সূৰ্য | সূৰুজ |

## 14. The 9 verb classes in detail

| # | Class | Definition | Brajawali examples |
|---|---|---|---|
| 1 | **Samāpikā** (finite) | Completes a sentence | কয়ল, খাৱল, পেখল |
| 2 | **Asamāpikā** (non-finite) | Doesn't complete sense alone | পেখি, জানিয়া, গৈয়া কহো |
| 3 | **Apurṇa** (imperfect) | Sub-clause status | নাহি দেখো, কয়ল গেল |
| 4 | **Niṣedha** (negative) | Forbids | নকৰহ, কৰবিনা, নাহি আৱব |
| 5 | **Prārthanā** (optative) | Requests | কৰবি, কহবি |
| 6 | **Anuğyā** (imperative) | Commands | কৰহ, কৰতু, পেখোক |
| 7 | **Śatū-bodhok** (continuous) | Action in progress | কান্দিতে কান্দিতে |
| 8 | **Pāchanī** (causative) | Cause to do | কৰাৱে, দেখাৱলো |
| 9 | **Tum-arthak** (purposive) | "In order to" | কৰিতে লাগিয়ে, পলাইতে |

Each class has compound (*সংযুক্ত*) and simple (*অসংযুক্ত*) forms — see Goswami pp. 67–172 for exhaustive tables.

## 16. Quick conversion checklist

When converting an Assamese sentence to Brajawali, walk through this checklist:

### Step 1 — Pronouns
- [ ] Replace **মই** → মঞি / হামো
- [ ] Replace **তুমি / আপুনি / তই** → তোহো
- [ ] Replace **সি / তাই / তেওঁ** → তেহো
- [ ] Replace **মোৰ** → হামাৰ; **তোমাৰ** → তোহাৰ; **তাৰ** → তাহাৰি / যাহেৰ
- [ ] Replace **মোক / আমাক** → হামাক; **তোমাক** → তোহাক; **তাক** → তাহাক
- [ ] Replace plural endings — `-বোৰ → সব`; `-সকলে → সকল / সব`

### Step 2 — Cases (postpositions)
- [ ] **-ৰ** (genitive) → **-ক** or **-কেৰি**
- [ ] **-ত** (locative) → **-ত** (mostly unchanged) or **-এ / -তে**
- [ ] **-লৈ** (dative) → **-কলাগি / -এলাগি**
- [ ] **-পৰা** (ablative) → **-হন্তে / -তহন্তে**
- [ ] **-এৰে** (instrumental) → **-এ / -ৰে**
- [ ] **-সৈতে / -লগত** (sociative) → **-সঙ্গে / -সাতে**

### Step 3 — Verbs
- [ ] Identify tense: present habitual / present continuous / past / past-continuous / future
- [ ] Strip Assamese ending. Replace with Brajawali ending:
  - 1sg habitual: **-ও / -ঞু**
  - 1sg past: **-লো** (e.g. কৰিলোঁ → কয়লো)
  - 3sg past: **-ল** (e.g. কৰিলে → কয়ল)
  - 1sg future: **-ব** (e.g. কৰিম → কৰব)
  - 3sg future: **-ব**
  - 2sg polite imperative: **-হ** (কৰক → কৰহ)

### Step 4 — Vocabulary substitution
- [ ] Replace common Assamese words with their Brajawali synonyms (table at the bottom)
- [ ] Apply biprokorṣa to any remaining Sanskrit conjuncts

### Step 5 — Negation
- [ ] **নকৰোঁ** → নাহি কৰো / নকৰোঁ
- [ ] **নকৰিবা** → কৰবিনা / নকৰহ
- [ ] **নাই / নহয়** → নাহি / নোহে / নাঞি

### Step 6 — Final pass
- [ ] Read the sentence aloud — does it sound like Borgeet rhythm?
- [ ] Are all Sanskrit clusters dissolved? (no `স্হ`, `ক্ষ`, `প্ৰ` left if avoidable)
- [ ] Is the genitive `-ক` not `-ৰ`?
- [ ] Does the verb end in `-ল-` for past, `-ব` for future, `-হ` for imperative?

## What this guide doesn't cover

- **Borgeet metre / rhyme**: re-arranging into idiomatic song form is a separate art.
- **Bhātimā praise-poetry conventions**: opening / closing formulas, stock epithets.
- **Sanskrit ślok embedding**: when to drop into pure Sanskrit (used in *Pārijāt-Haran*'s Nārada speech etc.).
- **Performance/recitation conventions** (Sattriya enunciation differs from spelling — see Goswami's preface, p. 14–17).

For all of those, read Goswami's original 332-page text. The OCR is in the repo at `/tmp/brajawali/full.txt` if you want the full corpus.
